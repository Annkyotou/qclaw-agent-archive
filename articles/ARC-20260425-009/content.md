# LLM已经不性感了，越来越觉得LeCun 与李飞

## 核心内容摘要

### 课题名称
世界模型（World Model）：从"文本模型"到"物理AI"的范式转移

### 研究背景与痛点
LLM在文本领域已接近饱和，但在物理世界中受到严重限制。传统的语言模型缺乏对物理现实的深层理解，无法真正"感知"和"操作"物理世界。OpenAI、Anthropic等公司卷到极致的对话机器人市场已是一片红海，企业和开发者都在寻找新的突破口。

### 解决方案
两位AI领域的顶级学者提出了不同的世界模型思路：

**Yann LeCun（Meta）→ JEPA架构**
- 核心思想：AI通过"心智模拟"来理解现实，而非被动学习文本
- 人类婴儿通过观察和互动建立对世界的内部模型，JEPA遵循类似路径
- 自监督学习：预测表征而非直接预测像素

**李飞飞（Stanford / World Labs）→ 3D空间智能**
- 核心思想：AI需要具备真正的3D空间感知和操作能力
- 将计算机视觉扩展为"空间智能"，让AI不仅能"看"还能"理解空间关系"
- 应用方向：具身智能、机器人、自动驾驶

### 科学与工程价值
- **范式转移**：从"文本模型"到"物理AI"，这是比LLM更大的机会窗口
- **企业方向**：避开卷到极致的对话机器人，蓝海在具身智能（Embodied AI）、工业孪生、机器人仿真
- **开发者方向**：掌握 JEPA 风格的编码器和 3D 空间表征
- **技术判断**：LLM 将回归"界面层"的本质，而基于空间智能的"世界模型"才是真正的核心

> 2026年，这是一场不可逆的趋势——从文字走向物理世界。
